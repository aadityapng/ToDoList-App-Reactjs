ToDoList-App-Reactjs
This project is a simple To-Do List application built using React and Vite.js.

Getting Started
To run the project locally, follow these steps:

Prerequisites
Make sure you have Node.js (version 14 or higher) and Git installed on your computer.

Node.js
Git
Clone the Repository
Clone this repository to your local machine:

sh

git clone https://github.com/aadityapng/ToDoList-App-Reactjs.git
cd ToDoList-App-Reactjs
Install Dependencies
Install project dependencies using npm:

sh
Copy code
npm install
Run the Project
Start the development server using Vite:

sh
Copy code
npm run dev
The application will be available at http://localhost:5173 in your web browser.

Available Scripts
In the project directory, you can run:

npm run dev: Starts the development server.
npm run build: Builds the project for production.
npm run serve: Serves the production build.
